KIRITO AI - Ultimate Version
Welcome to the repository for KIRITO AI, your advanced desktop assistant designed to bring a Siri-like experience with a realistic 3D interface right to your computer! This project is a comprehensive AI companion built with Python, offering a wide range of functionalities to manage your digital life.

üöÄ Getting Started
Ready to bring KIRITO AI to life? Here's how to get started:
Prerequisites

For running the executable: A Windows operating system is recommended.
For running from source:
Python 3.x installed.
Required libraries (e.g., speech_recognition, pyttsx3, pywhatkit, wikipedia, pyjokes, webview, pyautogui). It's highly recommended to install these using pip:pip install speech_recognition pyttsx3 pywhatkit wikipedia pyjokes pywebview pyautogui





Installation & Running
Option 1: Running the Executable (Easiest)

Download kirito_ultimate.exe from the repository.
Double-click the executable to launch KIRITO AI.
(Optional) The run_kirito.bat file might also be used to launch the assistant.

Option 2: Running from Source Code

Clone the repository:git clone https://github.com/kriss2012/kirito1.0.git
cd kirito1.0


Install the necessary Python libraries (as mentioned in prerequisites).
Run the main script:python kirito_ultimate.py

This will launch the assistant, opening a browser window or a dedicated pywebview window.

üí° How to Use KIRITO AI
Once KIRITO AI is running, you can interact with it in two primary ways:

Voice Commands:
Say a trigger word (e.g., "Kirito", "Kiroto", "Teri to").
Follow up with your command (e.g., "Kirito, play some music", "Kirito, what's the time?").


Text Input:
Use the text input field in the 3D interface.
Type your commands and press Enter or click "SEND".
